# GreenRide – Carpooling Platform

## Περιγραφή
Το GreenRide είναι μια web εφαρμογή carpooling που επιτρέπει σε χρήστες να
δημιουργούν και να μοιράζονται διαδρομές, μειώνοντας το κόστος και το
περιβαλλοντικό αποτύπωμα των μετακινήσεων.

## Τεχνολογίες
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- H2 Database
- Thymeleaf
- REST API (JSON)
- JWT Authentication
- Swagger / OpenAPI

## Ρόλοι Χρηστών
- USER: δημιουργία & κράτηση διαδρομών
- ADMIN: διαχείριση χρηστών & στατιστικά
  
## Λειτουργικότητα Χρηστών
- Δημιουργία διαδρομής από χρήστη-οδηγό (αφετηρία, προορισμός, ώρα, διαθέσιμες θέσεις).
- Αναζήτηση διαθέσιμων διαδρομών από επιβάτες.
- Κράτηση θέσης σε διαδρομή.
- Προβολή κρατήσεων χρήστη (My Bookings).
- Εμφάνιση οδηγού (driver) για κάθε κράτηση.
- Ακύρωση κράτησης σύμφωνα με τους κανόνες του συστήματος.

  ## Επιχειρησιακοί Κανόνες
- Δεν επιτρέπεται κράτηση διαδρομής χωρίς διαθέσιμες θέσεις.
- Ο ίδιος χρήστης δεν μπορεί να κάνει διπλή κράτηση στην ίδια διαδρομή.
- Η ακύρωση κράτησης από απλό χρήστη επιτρέπεται έως 10 λεπτά πριν την αναχώρηση.
- Ο διαχειριστής (ADMIN) μπορεί να ακυρώνει οποιαδήποτε κράτηση.


## Εκτέλεση Εφαρμογής

## Προαπαιτούμενα
- Java 17+
- Maven

## Build
mvn clean package

## Εκτέλεση
mvn spring-boot:run

## URLs
Web UI: http://localhost:8080

Η εφαρμογή περιλαμβάνει τις παρακάτω HTML σελίδες:

- home.html: δημόσια αρχική σελίδα
- login.html: σύνδεση χρηστών
- register.html: εγγραφή νέων χρηστών
- profile.html: προβολή στοιχείων χρήστη και ρόλου
- rides.html: δημιουργία και αναζήτηση διαδρομών, κράτηση θέσης
- bookings.html: προβολή κρατήσεων χρήστη και οδηγού διαδρομής

Swagger UI: http://localhost:8080/swagger-ui.html

## Authentication
Web UI: session-based login
REST API: JWT tokens

## Ροή Χρήστη
Μετά το login, ο χρήστης οδηγείται στη σελίδα Profile και μπορεί να μεταβεί στις σελίδες Rides και My Bookings για τη διαχείριση διαδρομών και κρατήσεων.  
Η πρόσβαση στις σελίδες αυτές επιτρέπεται μόνο σε αυθεντικοποιημένους χρήστες.


## Εξωτερική Υπηρεσία
Η εφαρμογή χρησιμοποιεί OpenWeather API για την εμφάνιση καιρού
κατά την ημέρα της διαδρομής.

Σε περίπτωση αποτυχίας της εξωτερικής υπηρεσίας, η εφαρμογή
συνεχίζει να λειτουργεί κανονικά.
